== Description ==
The plugin

Using this plugin you can create shortcode maps and set them in your posts or pages.

Используя данный плагин, вы сможете создавать шорткоды карт и размещать их на своем сайте.



== Installation ==
* Use standart WordPress instalator
* Or extract archive into "/wp-content/plugins/" folder
